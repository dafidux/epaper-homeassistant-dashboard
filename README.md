# epaper-homeassistant-dashboard
A project to create a MQTT interface/dashboard for use with homeassistant, featuring a RPi Zero 2 W and Waveshare 2.13 Touch Epaper display


REQUIREMENTS:

You need the libraries supplied by Waveshare, which you can find here:
https://www.waveshare.com/wiki/2.13inch_Touch_e-Paper_HAT_Manual#Raspberry_Pi

After following these instructions, place the main.py in the /python/examples folder.

You can configure password, ip, mqtt message etc. all in the python script
